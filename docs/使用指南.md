## 软件介绍

**启程典** 是一款桌面场景化启动工具，可以帮你：

- 📁 把常用的应用、文件、文件夹、网址分类整理
- 🚀 一键启动整个场景，省去逐个打开的麻烦
- 🎯 让你的工作桌面井井有条

### 基本概念

| 概念 | 解释 |
|------|------|
| **场景** | 一个"工作台"，可以放多个要用的东西。比如"办公场景"可以放 Word、Excel、邮件 |
| **启动项** | 场景里的具体内容，可以是应用、文件、文件夹或网址 |
| **启动** | 点击一下，打开整个场景里所有的东西 |

## 下载与安装

### 第一步：下载安装包

1. 从发布页面下载 `RiseDock_1.0.0_x64-setup.exe` 文件
2. 文件会保存到浏览器的下载目录（通常是"下载"文件夹）

### 第二步：运行安装程序

1. 打开"下载"文件夹
2. 找到 `RiseDock_1.0.0_x64-setup.exe` 文件
3. **双击**这个文件运行安装程序

> 💡 如果弹出"Windows 安全性"提示，问你是否允许，点击"仍要运行"或"确定"

### 第三步：完成安装

1. 安装向导会出现
2. 选择安装位置（默认即可，直接点"下一步"）
3. 等待安装完成（进度条走完）
4. 点击"完成"

### 第四步：启动软件

1. 安装完成后，软件会自动启动
2. 桌面上会出现一个蓝色图标的快捷方式
3. 以后双击这个图标就能打开软件

## 首次使用 - 创建你的第一个场景

### 界面认识

打开软件后，你会看到：

```
┌─────────────────────────────────────────────────────────┐
│  启程典                                        [_][□][X]│  ← 标题栏
├────────────┬────────────────────────────────────────────┤
│            │                                            │
│  场景列表   │         中间空白区域（右侧）                │
│            │         （会显示当前场景的内容）             │
│  [+新建场景]│                                            │
│            │                                            │
│            │                                            │
│            │                                            │
├────────────┴────────────────────────────────────────────┤
│  [帮助]                              [授权管理]          │  ← 底部按钮
└─────────────────────────────────────────────────────────┘
```

### 步骤 1：点击新建场景

1. 在左侧找到 **"+ 新建场景"** 按钮
2. 点击它

### 步骤 2：给你的场景起个名字

1. 点击后，会自动创建一个叫"新场景"的项目
2. 如果想改名：
   - 把鼠标移到场景名称上
   - 点击出现的 ✏️ **铅笔图标**
   - 输入你喜欢的名字（比如"工作"、"游戏"、"日常"）
   - 按键盘上的 **Enter** 键确认

### 步骤 3：选中场景

1. 点击场景名称
2. 右侧区域会显示这个场景的内容
3. 现在是空的，可以开始添加启动项了

## 添加启动项（应用/文件/文件夹/网址）

### 认识添加界面

在场景编辑页面，右上角有一个 **"+ 添加"** 按钮。

### 步骤 1：点击添加按钮

1. 在场景页面右上角找到 **"+ 添加"** 按钮
2. 点击它

### 步骤 2：选择要添加的内容类型

会弹出一个表单，有5种类型可选：

| 图标 | 类型 | 说明 | 举例 |
|------|------|------|------|
| 🖥️ | 应用 | 电脑上的程序 | 微信、浏览器、游戏 |
| 📄 | 文件 | 具体的文件 | 文档、图片、视频 |
| 📁 | 文件夹 | 文件夹 | 下载文件夹、项目文件夹 |
| 🔗 | 网址 | 网页链接 | 百度、邮箱、公司网站 |
| 🖼️ | 图片 | 图片文件 | 壁纸、参考图 |

### 步骤 3：根据类型填写信息

#### 🖥️ 添加应用（程序）

1. 在"名称"框输入名字（可以随便起，方便你认识就行）
2. 点击 **"浏览"** 按钮
3. 在弹出的窗口中找到你要添加的程序
   - 常用程序一般在 `C:\Program Files` 或 `C:\Program Files (x86)`
   - 或者去桌面找快捷方式
4. 选中程序后点击"打开"
5. 路径会自动填入
6. 点击 **"添加"** 按钮完成

#### 📄 添加文件

1. 输入名称
2. 点击 **"浏览"**
3. 找到要添加的文件（可以是 Word、Excel、图片等任何文件）
4. 选中后点击"打开"
5. 点击 **"添加"**

#### 📁 添加文件夹

1. 输入名称
2. 点击 **"浏览"**
3. 找到要添加的文件夹
4. 点击 **"选择文件夹"**（注意不是"打开"！）
5. 点击 **"添加"**

#### 🔗 添加网址

1. 输入名称（比如"百度"）
2. 在路径框直接输入网址（带 https://），比如：`https://www.baidu.com`
3. 点击 **"添加"**

#### 🖼️ 添加图片

1. 输入名称
2. 点击 **"浏览"**
3. 找到图片文件（支持 png、jpg、gif 等格式）
4. 选中后点击"打开"
5. 点击 **"添加"**

### 步骤 4：查看添加结果

添加成功后，会在场景列表中看到新增的项目。

## 管理启动项（编辑/删除/排序）

### 查看启动项列表

在场景页面中间区域，会显示所有添加的启动项卡片。

### 编辑启动项

如果想修改某个启动项的名字或路径：

1. 在启动项卡片上**右键点击**
2. 在弹出的菜单中选择 **"编辑"**
3. 修改名称或点击"浏览"重新选择文件
4. 点击 **"保存"** 按钮

### 删除启动项

如果不需要某个启动项了：

1. 在启动项卡片上**右键点击**
2. 在弹出的菜单中选择 **"删除"**
3. 确认删除即可

### 拖拽排序

调整普通启动项的显示顺序（置顶项目不支持拖拽排序）：

1. 鼠标移到启动项卡片上
2. 按住左键不放
3. 拖动到想要的位置
4. 松开左键放下

> 💡 置顶项目会显示在顶部，按添加顺序固定显示，不支持拖拽排序

## 场景管理（新建/重命名/删除）

### 新建更多场景

1. 在左侧点击 **"+ 新建场景"**
2. 自动创建新场景并选中

### 重命名场景

1. 把鼠标移到场景名称上
2. 点击出现的 **✏️ 铅笔图标**
3. 输入新名称
4. 按 **Enter** 确认

### 删除场景

> ⚠️ 删除后无法恢复！

1. 把鼠标移到场景名称上
2. 右键点击场景名称
3. 在弹出的菜单中选择 **"删除场景"**
4. 确认删除

### 切换场景

- 只需点击左侧的场景名称即可切换
- 右侧会显示对应场景的内容

## 启动场景

### 方式一：点击单个启动项

在场景页面，点击任意启动项卡片的 **"启动"** 按钮可以单独启动它。

### 方式二：右键启动

1. 在启动项卡片上右键点击
2. 选择 **"启动"**

### 方式三：置顶功能

把常用的启动项**置顶**，会显示在列表最上方，方便快速找到。

1. 右键点击启动项
2. 选择 **"置顶"**
3. 置顶的项目会显示在顶部

### 最小化和关闭

- **最小化按钮（—）**：正常最小化到任务栏
- **关闭按钮（X）**：点击后会弹出确认框，选择"最小化到托盘"或"退出程序"

> 💡 点击关闭按钮后，软件会在后台托盘运行，双击托盘图标可以恢复窗口

## 未授权限制说明

### 什么是授权？

授权是软件的"解锁钥匙"。未授权时，软件有部分功能限制。

### 未授权可以做什么？

✅ **可以做的事情：**

| 功能 | 说明 |
|------|------|
| 创建场景 | 最多 1 个场景 |
| 添加启动项 | 每个场景最多 3 个启动项 |
| 编辑启动项 | 修改名称、路径 |
| 删除启动项 | 移除不需要的项目 |
| 拖拽排序 | 拖拽调整顺序 |
| 切换场景 | 在不同场景间切换 |
| 快速搜索 | Ctrl+K 搜索启动项 |

❌ **不能做的事情：**

| 功能 | 说明 |
|------|------|
| 创建多个场景 | 只能有 1 个场景 |
| 添加更多启动项 | 超过 3 个就无法添加了 |

### 提示信息

当操作被限制时，会弹出提示：
- "未授权版本最多创建 1 个场景"
- "未授权版本每个场景最多添加 3 个启动项"

## 激活授权

### 什么是授权码？

授权码是一串字母和数字的组合，用于解锁软件的付费功能。

### 获取授权码

请联系软件开发者购买授权，获取授权码。

### 输入授权码

1. 点击右下角的 **"授权管理"** 按钮
2. 在弹出的窗口中，你会看到输入框
3. 把授权码粘贴或输入进去
4. 点击 **"激活"** 按钮
5. 如果授权码正确，会看到"激活成功"提示

### 授权类型

| 类型 | 有效期 | 价格 | 说明 |
|------|--------|------|------|
| 月度授权 | 1个月 | 实惠 | 适合短期使用 |
| 季度授权 | 3个月 | 推荐 | 性价比高 |
| 年度授权 | 12个月 | 划算 | 长期使用首选 |
| 永久授权 | 永久 | 一次付费 | 永久使用，终身更新 |

## 授权版本功能

激活授权后，所有功能解锁！

### ✅ 创建多个场景

不受限制地创建任意数量的场景。

### ✅ 添加更多启动项

每个场景可以添加无限多个启动项。

### ✅ 置顶功能

将常用的启动项置顶，方便快速找到。

**使用方法：**
1. 右键点击启动项
2. 选择 **"置顶"**
3. 置顶的项目会显示在列表顶部

### ✅ 快速搜索

使用 Ctrl+K 快捷键快速搜索启动项。

**使用方法：**
1. 按下 **Ctrl + K**
2. 输入关键词搜索
3. 选择结果后高亮显示

## 常见问题

### Q：软件打不开怎么办？

**A：**
1. 检查是否安装成功
2. 尝试重启电脑
3. 重新安装软件

### Q：添加应用时找不到程序？

**A：**
- 常用程序一般在 `C:\Program Files` 或 `C:\Program Files (x86)` 文件夹
- 也可以去桌面找快捷方式，右键 → "打开文件所在的位置"

### Q：启动项打不开？

**A：**
1. 检查路径是否正确
2. 如果文件被移动或删除了，需要重新编辑路径

### Q：授权码激活失败？

**A：**
1. 检查授权码是否输入正确（注意大小写）
2. 确保网络连接正常
3. 联系开发者获取帮助

### Q：如何卸载软件？

**A：**
1. 打开 Windows 的"设置"
2. 点击"应用"
3. 找到"启程典"或"RiseDock"
4. 点击"卸载"

### Q：软件最小化后找不到了？

**A：**
- 如果点击的是最小化按钮（—），软件会在任务栏显示
- 如果点击的是关闭按钮（X），软件会最小化到屏幕右下角的系统托盘
- 找到托盘图标（蓝色小图标），双击恢复窗口

### Q：点击关闭按钮会怎样？

**A：**
- 点击关闭按钮（X）后，会弹出确认框
- 选择 **"最小化到托盘"**：软件在后台运行，可在托盘恢复
- 选择 **"退出程序"**：完全关闭软件
