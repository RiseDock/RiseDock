# 启程典（RiseDock）开发路线图

> 最后更新：2026-04-25

## 当前状态

| 项目 | 信息 |
|------|------|
| 当前版本 | v1.0.3.1（紧急修复版） |

- 启动执行（应用/文件/文件夹/网址/延迟启动）

- 全局快捷键唤起
- 授权系统（机器码采集 → 激活码验证 → 本地持久化）

- 置顶固定
- 批量导入/导出 JSON
- 自定义标题栏

## v1.1.0 — 授权体验优化 + 自动更新

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 自动更新 | `@tauri-apps/plugin-updater` + GitHub Releases，免签名方案 | P0 |
| 到期前提醒 | 7天/3天/1天三级提醒弹窗 | P1 |
| 授权续期 | 到期前可输入新授权码续期 | P1 |
| 授权信息导出 | 导出机器码+授权状态，方便换机迁移 | P2 |

**预计工作量**：2-3天

### 自动更新方案（已确认）

- 使用 `@tauri-apps/plugin-updater` 插件
- 后端对接 GitHub Releases API
- 不购买代码签名证书，采用免签名方案
- 用户首次启动时检查更新，后台静默下载，下次启动提示安装

## 推荐执行顺序

### 本周
1. **v1.1.0 自动更新** — 方案已确认，零成本可用，优先让用户能自动升级

6. **v1.3.0 云同步** — 等用户量起来再投入
