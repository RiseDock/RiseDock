# Icons

This directory contains icons for the application.

For Tauri 2.0, you need the following icons:
- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)

You can use online tools like [Favicon.io](https://favicon.io/) or [RealFaviconGenerator](https://realfavicongenerator.net/) to generate these from your SVG icon.
